- **欢迎Pull requests和Issues**

为了方便**Pull requests**，从而共享自己的解题思路和代码实现，我专门写了篇教程[在Github中进行pull request操作](https://wxler.github.io/2021/02/25/170421/)。当然，大家也可以成为该项目的Collaborator，一起管理本项目，具体操作方法也在这篇教程中。
